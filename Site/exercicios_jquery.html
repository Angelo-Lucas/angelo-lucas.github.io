jQuery

jQuery é uma biblioteca criada para simplificar a forma de codificação de JavaScript nos documentos web.
"Write less, do more."

Adicionar jQuery ao seu WebSite

Existem várias maneiras de começar a usar o jQuery em seu site. Você pode:

  Baixar a biblioteca jQuery de jquery.com;
  Incluir jQuery de um CDN, como o Google CDN.

Baixar jQuery

Existem duas versões do jQuery disponíveis para download:

  Versão de produção - Esta versão serve para ambientes de produção, é uma versão com o código em formato compactado, sem quebra de linhas e com o código sem comentários, possuindo apenas 15% do tamanho da que veremos a seguir;

  Versão de desenvolvimento - Esta versão é para desenvolvimento, é a versão não compactada da biblioteca e possui o código todo comentado. Sendo ideal para ambientes de desenvolvimento, por se integrar facilmente com as ferramentas e IDEs.


A biblioteca jQuery é um único arquivo JavaScript e você faz referência a ela com a tag HTML <script> (observe que a tag <script> deve estar dentro da seção <head>):

<head>
<script src = "jquery-3.3.1.min.js"> </script>
</head>

Dica: Coloque o arquivo baixado no mesmo diretório das páginas em que você deseja usá-lo.

jQuery CDN

Se você não quiser fazer o download e hospedar o jQuery, pode incluí-lo em um CDN. Para usar o jQuery do Google:
<head>
<script src = "https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"> </script>
</head>

Exemplos

Lembre-se de colocar o código no mesmo diretório das pastas "css" e "js" do jquery.

Exemplo 01 Visualizar

<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>JQuery</title>
  <script src="js/jquery-3.3.1.min.js"></script>
  <script>
  $(document).ready(function(){
    $("h1")
    .css("color", "#8e2323")
    .hide()
    .delay('2000')
    .fadeIn("slow")
    .text("Alterei o titulo");
  });
  </script>
</head>
<body>
  <h1>Meu primeiro exemplo</h1>
</body>
</html>
